The Quotes Management page allows staff to create, view, edit, and track all customer quotes. Quotes are displayed with key information including quote number, customer details, total amount, status, and validity period. Color-coded status badges make it easy to identify quotes at different stages (draft, sent, viewed, accepted, expired). Staff can quickly access detailed quote information or make edits as needed.

The User Management page is accessible only to administrators and provides tools to manage staff accounts. Admins can create new users, assign appropriate roles (Admin, Product Development, Marketing, Administration), and manage access permissions. The interface displays user status, last login time, and allows for account activation/deactivation. This ensures proper security and access control within the admin portal.
